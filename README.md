# FinStream Prime: Loan Processing Bottleneck & Efficiency Analysis

## Project Overview
This project analyzes loan processing efficiency using SQL Server, Python, Machine Learning, and Power BI.

## Technologies
- SQL Server
- Python (Pandas, Scikit-learn, Matplotlib)
- Power BI
- GitHub

## Features
- SQL Data Cleaning
- Workflow Analysis
- Feature Engineering
- Random Forest Prediction
- Power BI Dashboard
